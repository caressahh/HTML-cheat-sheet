HTML Cheat Sheet

  • Always begin an HTML document with <!DOCTYPE html> this tells the browser that you plan on using HTML in the document
  • An element is code inside brackets.
  • "Boiler plate code" is used to describe the basic HTML code required to begin creating a webpage. EX: <p> = paragraph.
  • <ul> = unordered lists - listed with bullet points.
  • <ol> = ordered lists - numbers each listed item.
  • <li> = used within <ul> or <ol> for each list item.
  • <h1> = makes text emphasized or bold.
  • target- opens linked page in a anew window. It requires a value of "_blank"
  • <img src="URL"/> is used to insert a photo link that will display the picture on the webpage.
  • alt used within img to describe the image: EX: alt= "text" before closing image.
  • Turn an image into a link by wrapping it with an anchor (aka <a>) EX:<a href="link" target="_blank"> <img src="# " alt="text"></a>
  • <br/> line break element (return)
  • 2 spaces of indentation are always used for elements nested within other elements-- DO NOT use tab.
  • <!--TEXT--> is used to add comment to HTML content.
